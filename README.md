Digital Clock

A real-time digital clock built with HTML, CSS, and JavaScript that displays the current time in hours, minutes, and seconds. The clock updates every second and supports a 12-hour or 24-hour format, making it a simple yet practical web project.

Features

Displays hours, minutes, and seconds dynamically.

Supports 12-hour and 24-hour formats.

Styled with responsive CSS for modern appearance.

Lightweight and easy to integrate into any web page.

Can be used as a learning project for beginners in JavaScript and DOM manipulation.
